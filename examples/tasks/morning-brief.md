# Morning Brief

Run this task every morning. Execute each step in order and compile the results into a single briefing.

## Steps

1. **Calendar** -- Pull today's events from all calendars. List each event with time, title, and location. Flag any conflicts or back-to-back meetings with no buffer.

2. **Email** -- Check inbox for unread messages received since yesterday's briefing. Categorize:
   - **Urgent** -- Requires response today
   - **FYI** -- Informational, no action needed
   - **Noise** -- Newsletters, promotions, automated notifications (archive or delete these)

3. **Open items** -- Read yesterday's daily journal. List any tasks that were in progress or flagged as "carry forward."

4. **Weather** -- Get today's weather forecast for the user's location. One sentence.

5. **Compile briefing** -- Present everything in a clean, scannable format:
   - Start with the single most important thing for today
   - Then calendar overview
   - Then email summary (counts by category, details only for urgent)
   - Then open items
   - End with weather

Keep the entire briefing under 500 words. If it's a light day, keep it under 200.
